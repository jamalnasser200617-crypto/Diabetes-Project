 Diabetes Prediction Project
This is a Machine Learning web application that predicts diabetes based on user input.

How it works
1. Preprocessed the dataset using StandardScaler.
2. Trained 4 different models (Logistic Regression, Random Forest, etc.).
3. Deployed using Streamlit.

Files
- `app.py`: Streamlit web interface.
- `diabetes_model.pkl`: Trained ML model.
- `scaler.pkl`: Scaler for data normalization.
